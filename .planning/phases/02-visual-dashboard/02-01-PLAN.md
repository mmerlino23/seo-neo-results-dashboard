4. `.header-stats-row` — Flexbox row for stat items. `display: flex; gap: 2rem; align-items: center; flex-wrap: wrap;`

5. `.stat-item` — Individual stat (number + label). Flex column, centered.

6. `.stat-number` — Large stat number. Font-size ~2rem, font-weight 800, color `--color-text-primary`.

7. `.stat-label` — Stat description text. Font-size 0.75rem, uppercase, letter-spacing 0.1em, color `--color-text-muted`.

8. `.badge` — Base campaign type badge. Inline-flex, padding 0.25rem 0.75rem, border-radius 9999px, font-size 0.75rem, font-weight 600.

9. `.badge-100-content` — Green badge using `--badge-100-content` as background at 15% opacity, text color at full.

10. `.badge-1-content-spin` — Blue badge using `--badge-1-content-spin` same pattern.

11. `.badge-cloud-posting` — Purple badge using `--badge-cloud-posting` same pattern.

12. `.badge-count` — Small count number next to badge. Font-weight 700, margin-left 0.5rem, color `--color-text-secondary`.

13. `.main-content` — Main content area below header. Max-width 1400px, margin auto, padding 2rem.

14. `.section-divider` — Subtle horizontal rule between sections. Height 1px, background gradient from transparent through `--color-border-subtle` back to transparent.
